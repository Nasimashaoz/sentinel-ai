# Security Policy

## Supported Versions
| Version | Supported |
|---|---|
| 1.x | ✅ |

## Reporting a Vulnerability
Do **not** open a public issue. Email the maintainer privately.
Expect acknowledgment in 48 hours, fix in 7 days.

## Important Notes for Users
- Sentinel AI reads log files — run with minimum required permissions
- Never expose the dashboard port (8080) to the public internet without authentication
- Store your `.env` file securely — it contains API keys
- Review the source code before running on production systems
