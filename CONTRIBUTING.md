# Contributing to Sentinel AI

Thank you! Sentinel AI grows with the community. 🛡️

## Most Wanted Contributions

### 📡 New Collectors
- AWS CloudTrail logs
- GCP Audit logs
- Azure AD sign-in logs
- Kubernetes audit logs
- Docker events
- Windows Event Log support

### 📱 New Alert Channels
- Discord webhooks
- Microsoft Teams
- Signal (via signal-cli)
- Line Notify
- Mattermost

### 🌐 Threat Intelligence Integration
- AbuseIPDB — check source IPs against threat feeds
- VirusTotal — hash/IP/domain lookups
- Shodan — check if attacking IP is in Shodan
- Greynoise — noise vs. targeted attack classification

### 🤖 AI Model Support
- Ollama (local LLM — zero API cost)
- OpenAI GPT-4o
- Google Gemini
- Mistral via API

### 📊 Dashboards
- Grafana dashboard template
- Prometheus metrics exporter
- ELK/OpenSearch integration

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/discord-alerts`
3. Make changes, test them
4. Open a PR with a clear description

## Questions?
Open an issue — all are welcome!
